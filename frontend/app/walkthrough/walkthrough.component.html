How to upload bot scripts locally
  1. Generate your code package: Click the generate button of the bot script you configured
  2. Navigate to your steam directory:
    • Windows: “C:\Program Files (x86)\Steam”
    • Mac: "Users/<USERNAME>/Library/Application Support/Steam
  3. Follow the corresponding path: if there is no "bots" folder, then create the folder
  4. Copy the downloaded zip into the bots folder above
  5. Extract the zip containing the code into this directory. If you do not have such software, you can download WinRar or 7zip
  6. It is not essential to delete the zip file, however you may wish to.

How to start a game with custom bot scripts
  1. Upload your bot scripts locally
  2. Launch Dota 2
  3. After clicking the play button, click create under the custom lobbies tab
  4. Click on the Edit button (small gear add icon)
  5. You may have to open the advanced settings tab
  6. You will see a tab labelled “Bot settings”:
    1. Although this is not essential, it is recommended that cheats are enabled.
    2. Ensure that the checkbox “fill empty slots with bots” is checked
    3. On any side(s) of the map where the custom bots are desired to be used, click the dropdown and select the local dev script option
