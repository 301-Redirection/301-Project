Getting Started

How to upload bot scripts locally:
  1. Generate the code package: Click the generate button of the bot script you configured
  2. Navigate to your steam directory:
    • Windows: “C:\Program Files (x86)\Steam” or “C:\Program Files\Steam”
    • Mac: "Users/YOUR_USERNAME/Library/Application Support/Steam"
  3. Follow the path: “steamapps\common\dota 2 beta\game\dota\scripts\vscripts\bots”. If the "bots" folder cannot be found, simply create one in the "vscripts" directory.
  4. Extract the zip containing the code into this directory


Start a game with custom bot scripts:
  1. Upload bot scripts locally (seen above)
  2. Launch Dota 2
  3. Click the Play button, and select Custom Lobby
  4. Click on the Edit button (small gear add icon)
  5. You will see a tab labelled “Bot settings”:
    1. Ensure that the checkbox “fill empty slots with bots” is checked
    2. For each side of the map (“radiant” and “dire”), select the bot difficulty
    3. On the side(s) of the map where the custom bots are desired to be used, click the dropdown and select the local script option
  6. Start playing!
